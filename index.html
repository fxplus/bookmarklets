Helpful Bookmarklets

these bookmarklets* are helpers for use with fx plus - for example take you from an imdb page to that film on the library catalogue, if we have it.
  • imdb: film to library search
  • goodreads: book to library search

Update

This bookmarklet looks up items from:

  • goodreads
  • rottentomatoes
  • imdb
  • suggestmemovie
  • wikipedia.org
  • flixster.com
  • check the library

Firefox users:
For some reason it seems this bookmarklet will not drag to the bookmarks bar in firefox (too long?). If you rightclick the link you can howver 'Bookmark this link'

if its useful let me know, I can always make a better one or another one for a similar service

* to use a bookmarklet like this just drag the link into your bookmarks, then click it when you want to use it on a page
